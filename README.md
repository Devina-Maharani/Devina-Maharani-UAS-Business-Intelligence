# Business Intelligence â€“ PriceRunner Dataset

Proyek ini merupakan pengembangan sistem Business Intelligence end-to-end menggunakan dataset PriceRunner. Sistem dirancang untuk mendukung analisis data dan prediksi tren pada platform e-commerce melalui proses pengolahan data yang terstruktur dan terotomatisasi.

Seluruh alur kerja dimulai dari proses ETL (Extract, Transform, Load) yang diotomatisasi menggunakan Apache Airflow. Data kemudian diproses, dibersihkan, dan dimodelkan menggunakan Python untuk menghasilkan prediksi, yang selanjutnya divisualisasikan dalam bentuk grafik perbandingan data aktual dan data prediksi melalui dashboard interaktif berbasis Django.

## Fitur Utama
- ETL otomatis menggunakan Apache Airflow  
- Pembersihan dan pengolahan data menggunakan Python  
- Pemodelan prediktif untuk analisis tren  
- Dashboard interaktif berbasis Django  
- Visualisasi perbandingan data aktual dan prediksi  

## Tech Stack
- Python  
- Apache Airflow  
- Django  
- MySQL
